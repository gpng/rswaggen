# RSwag Schema Generator

Generate RSwag schema from JSON input

## Table of Contents

- [Usage](#usage)
- [Maintainers](#maintainers)

## Usage

1. Clone this repo

1. Install dependancies

   ```
   yarn
   ```

1. Run app

   ```
   yarn dev
   ```

   or if you have vercel-cli installed,

   ```
   vercel dev
   ```
